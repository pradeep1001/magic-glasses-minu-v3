# Minu's Magic Glasses — Complete Dialogue List (v2)
# ==================================================
# TWO VOICES:
#   1. NARRATOR — Teaches the child, explains concepts, guides actions
#   2. MINU — Reacts emotionally, expresses confusion/surprise/gratitude
#
# PEDAGOGY:
#   - Minu is an alien who sees the world as RAW NUMBERS (0-255)
#   - She CANNOT see images like we do — only pixel values
#   - The child helps calibrate her "magic glasses" so she can see properly
#   - The Narrator teaches the child HOW to help Minu
#   - Minu reacts with genuine emotion as she starts to "see" things
#
# VOICE DIRECTION:
#   NARRATOR: Warm, friendly adult (think PBS/Blue's Clues host)
#     - Speaks clearly and slightly slowly for ages 5-8
#     - Encouraging, patient, enthusiastic
#     - Never condescending
#
#   MINU: Childlike alien voice (higher pitch, slightly quirky)
#     - Genuinely confused when she sees numbers
#     - Excited and amazed when she starts to "see"
#     - Grateful when the child helps her
#     - A bit naive and innocent
#
# GENERATION GUIDANCE:
#   - Format: MP3, 44.1kHz
#   - Normalize all files to -3dB peak
#   - Keep volume consistent across all files
#   - Speed: Slightly slower than normal conversation
